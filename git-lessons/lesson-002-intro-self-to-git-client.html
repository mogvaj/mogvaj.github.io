DCN: How to Git

Lesson 2 - Introduce Yourself to Your Installation of Git

  • Green
    • Start up Git bash:
      • Click on the "Start" button
      • Scroll down to the "Git" folder
      • Expand the "Git" folder and verify that "Git bash" brings up a UNIX-ey looking window
    • Tell your installation of Git who you are by typing:
        • $ git config --global user.name "YourFirstName YourLastName" $ git config --global user.email "your@email.com"
    • You can verify those settings:
        • $ git config --get user.name YourFirstName YourLastName $ git config --get user.email your@email.com $
