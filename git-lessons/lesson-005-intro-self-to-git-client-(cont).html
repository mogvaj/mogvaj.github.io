DCN: How to Git

Lesson 5 - Introduce yourself to your Git client on your workstation

  • Open File Manager
  • Navigate to a folder where you would like to work on development projects
    • (For example, I have a sub-folder within my "Documents" folder called "dev".
    • Whenever I work on a development project, it happens within a sub-folder in the "dev" folder.)
  • Create a sub-folder named "dev"
  • Enter into the "dev" folder
  • Right click somewhere in the file list pane of the window
  • Choose "Git Bash Here"
  • You'll see a UNIX-ey looking shell prompt
  • Type "git status" and press enter
  • You should see an error:
      • $ git status fatal: not a git repository (or any of the parent directories): .git $
  • This is true --- you don't have a git repository set up here
  • DONE!
      • (We're now ready to clone repositories to your workstation!)
