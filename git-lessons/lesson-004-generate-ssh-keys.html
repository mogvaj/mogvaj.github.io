Lesson 4 - Create SSH keys

  • Note 1: We probably don't need to do this. If things don't work out in later lessons, you might need to do this....
  • Note 2: SSH keys will need to be generated for each workstation you work from (i.e., your desktop, your laptop, etc.)
    • Create SSH Keys:
      • Open a Windows Command Prompt (i.e., the "DOS prompt")
      • Generate an RSA public key:
          • C:\Users\YourWindowsUsername\ssh-keygen Generating public/private rsa key pair. Enter file in which to save the key (C:\Users\flast/.ssh/id_rsa): {{press enter to accept the default}} ... {{Enter a passphrase twice}}
      • When the key has been generated, close the command prompt window
      • Use File Manager to navigate to c:\Users\YourWindowsUsername
      • There should be a hidden folder named ".ssh"
        • If not, make sure that you can view hidden files and folders in File Manager:
          • In File Manager...
          • Choose the "View" tab at top
          • In the ribbon that appears...
          • Put a checkmark in the "Hidden items" option
      • Enter the ".ssh" folder
      • There should be a file named "id_rsa.pub"...
      • Open that file with notepad (*not* MS Publisher...)
      • Select all text and copy it to the clipboard
      • Switch windows to a web browser
      • Go to https://github.com and login
      • Click your Profile Avatar
      • In the dropdown menu that appears, choose "Settings"
      • On the far left side-menu, click on "SSH and GPG Keys"
      • Paste the text of your SSH key into the topmost text area box
      • Give this key a title (maybe something like "My Laptop"
      • Click on the "Add key" button
      • DONE!
