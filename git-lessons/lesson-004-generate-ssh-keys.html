DCN: How to Git

Lesson 4 - Create SSH keys

  • Note 1: The following instructions are specific to the Green-side gitlab. These instructions also apply to the Red-side gitlab, however the URL is different, so keep that in mind.
  • Note 2: SSH keys will need to be generated for each workstation you work from (i.e., UNET physical at your desk, UNET virtual via Remote UNET, etc.)
    • Login to your UNET workstation (or Remote UNET, or UVDI, etc.)
    • Create SSH Keys:
      • Open a Windows Command Prompt (i.e., the "DOS prompt")
      • Generate an RSA public key:
          • C:\Users\flast\ssh-keygen Generating public/private rsa key pair. Enter file in which to save the key (C:\Users\flast/.ssh/id_rsa): {{press enter to accept the default}} ... {{Enter a passphrase twice}}
      • When the key has been generated, close the command prompt window
      • Use File Manager to navigate to c:\Users\flast
      • There should be a hidden folder named ".ssh"
        • If not, make sure that you can view hidden files and folders in File Manager:
          • In File Manager...
          • Choose the "View" tab at top
          • In the ribbon that appears...
          • Put a checkmark in the "Hidden items" option
      • Enter the ".ssh" folder
      • There should be a file named "id_rsa.pub"...
      • Open that file with notepad (*not* MS Publisher...)
      • Select all text and copy it to the clipboard
      • Switch windows to a web browser
      • Go to https://gitlab.fbi.gov and login
      • Click your Profile Avatar
      • In the dropdown menu that appears, choose "Edit profile"
      • On the far left side-menu, click on "SSH Keys"
      • Paste the text of your SSH key into the topmost text area box
      • Give this key a title (maybe something like "My UNET Workstation"
      • Click on the "Add key" button
      • DONE!
