DCN: How to Git

Lesson 1 - Installing Git

  • Green
    • Login to your UNET workstation (or Remote UNET, or UVDI, etc.)
    • Click on the "Start" button (or whatever that thing is called at the lower left of the desktop)
    • Start entering "\\fbi.gov\public\software"
    • Scroll to the folder for "GIT" and navigate into it
    • Double click on the installation file
    • For the most part, you can accept defaults, but make sure:
      • On the dialog titled "Choosing HTTPS transport backend":
        • Choose the radio button next to "Use the native Windows Secure Channel library"
    • To verify that it is working:
      • Start up File Manager
      • Right click somewhere in the file listing pane
      • Choose "Git Bash Here"
      • A UNIX-ey looking window should come up
    • DONE!
