DCN: How to Git

Lesson 3 - Introduce Yourself to Gitlab

  • Start up a browser and browse to https://github.com
  • If you don't already have a GitHub account:
    • Choose "Sign up" and...sign up
    • Return to the top of these instructions
  • If you already have a GitHub account:
    • Choose "Sign in" and...sign in
    • Click on your "Profile Avatar" button in the upper right corner
    • Choose "Settings"
    • On the left side menu, choose "Developer Settings"
    • Create a Personal Access Token:
      • On the left side menu, click on "Personal Access Tokens"
      • On the drop-down that appears, choose "Tokens (classic)"
      • Click on the button at top labeled "Generate new token"
      • On the drop-down that appears, choose "Generate new token (classic)"
      • In the "Note" field, type something like "20250731-personal-access-token"
      • Set the expiration date per your preference
      • Click the checkboxes next to all the Scope settings (for training purposes)
      • Click the "Generate token" button
      • The access token will be shown near the top of the page
      • Copy the new personal access token to the clipboard
      • Create a text file wherever you keep documents (i.e., your "Documents" folder) and name it "git-information.txt"
      • In the "git-information.txt" file, enter the following:
          • Gitlab Personal Access Tokens:
    ghp_notarealpersonalaccesstoken
      • Save that file (we'll use that Personal Access Token later)
      • DONE!
