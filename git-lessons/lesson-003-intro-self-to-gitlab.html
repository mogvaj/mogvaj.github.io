DCN: How to Git

Lesson 3 - Introduce Yourself to Gitlab

  • Note: The following instructions are specific to the Green-side gitlab. These instructions also apply to the Red-side gitlab, however the URL is different, so keep that in mind.
    • Login to your UNET workstation (or Remote UNET, or UVDI, etc.)
    • Start up a browser and load https://gitlab.fbi.gov
    • Click on the button to "Sign in with [FBI Federation Services]"
    • Enter whatever you have to in order to get in
    • Click on your "Profile Avatar" button in the upper right corner
    • Choose "Edit profile"
    • Create a Personal Access Token:
      • On the left side menu, click on "Access Tokens"
      • In the "Name" field, type something like "UNET Workstation" or "Remote UNET" (depending on where you are)
      • Leave the expiration date blank
      • Click the checkbox next to all the Scope settings (because, why not?)
      • Click the "Create personal access token" button
      • Copy the new personal access token to the clipboard
      • Create a text file wherever you keep documents (i.e., your "Documents" folder) and name it "git-information.txt"
      • In the "git-information.txt" file, enter the following:
          • Gitlab Personal Access Tokens:
   gsbo5Zrz7cfjdn5xtfs-
      • Save that file (we'll use that Personal Access Token later)
      • DONE!
