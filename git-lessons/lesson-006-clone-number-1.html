DCN: How to Git

Lesson 6 - Clone your first repository

  • Note 1: This one is a little longer...
  • Note 2: Sorry...
  • Start a browser and go to "https://github.com"
  • Login as necessary
  • Navigate to: https://github.com/mogvaj/hello-git
  • In the dropdown menu that appears, click "Your projects"
  • The publicly available "hello-git" repository appears
  • On the right side near the top there is a green button that says "Code"...click on it
  • A drop down menu will appear...
  • Copy the URL under "Clone with HTTPS" by clicking on the clipboard icon to the right of the URL
  • Done with the browser...you can close it now

  • Start up File Manager and navigate to your development folder (i.e., I called mine "development" under my "Documents" folder)
  • Right click somewhere in your development folder and choose "Git bash here"
  • Clone the repository:
    • Enter the following:
        • $ git clone {{DON'T press enter yet}}
    • Now right click in the bash window and paste the URL copied from Gitlab
        • $ git clone https://github.com/mogvaj/hello-git.git {{DON'T press enter yet}}
    • Now...left arrow over to the "g" of "gitlab"
    • Add "oauth2:" to the URL:
        • $ git clone https://oauth2:github.com/mogvaj/hello-git.git {{DON'T press enter yet}}
    • Now...open another instance of File Manager and navigate to your "git-information.txt" file
    • Open that file and add the following at the top:
        • To clone:
   git clone https://oauth2:{{personal access token}}@github.com/....git
    • That's the pattern for how to clone a repository from the Git bash command line (so you can look at it later)
    • Save "git-information.txt"
    • Copy your GitHub Personal Access Token to the clipboard:
        • To clone:
   git clone https://oauth2:{{personal access token}}@github.com/....git

Personal Access Tokens
   lerkdfoiuerkjhf
    • We can now close "git-information.txt"
    • Now...go back to the Git bash window and paste your personal access token after "oauth:"
        • $ git clone https://oauth2:lerkdfoiuerkjhfgithub.com/mogvaj/hello-git.git {{DON'T press enter yet}}
    • Next, put an "at" (@) sign after your token and press enter. Then the cloning magic starts:
        • $ git clone https://oauth2:lerkdfoiuerkjhf@github.com/mogvaj/hello-git.git Cloning into 'hello-git'... remote: Enumerating objects: 3, done. remote: Counting objects: 100% (3/3), done. remote: Compressing objects: 100% (2/2), done. remote: Total 3 (delta 0), reused 0 (delta 0), pack-reused 0 Unpacking objects: 100% (3/3), 275 bytes | 2.00 KiB/s, done. $
  • Verify you have cloned:
    • Type "git status" and press enter to see what you've done:
        • $ git status fatal: not a git repository (or any of the parent directories): .git $
    • Hey! Wait a minute! What's going on here?
        • $ ls -al total 36 drwxr-xr-x 1 flast 1049089 0 Oct 28 09:54 ./ drwxr-xr-x 1 flast 1049089 0 Oct 28 07:57 ../ drwxr-xr-x 1 flast 1049089 0 Oct 28 09:45 hello-git/ $
    • It's true. We aren't in a repository, yet. Git cloned that repository into a sub-directory called hello-git
    • Change to the "hello-git" sub-directory and do a "git status":
        • $ cd hello-git/ $ (main) git status On branch main Your branch is up to date with 'origin/main'. nothing to commit, working tree clean $ (main)
    • Now...you are in a repository. Not only that, you are currently using the "main" branch.
  • DONE!
